# 🧠 DevQuiz — Test Your Code. Level Up Your Mind.

> *An interactive coding quiz that challenges what you know — and how fast you know it.*

DevQuiz is a fast-paced, multi-language coding quiz app built to sharpen your programming fundamentals under pressure. Pick your language, pick your difficulty, and race the clock.

🔗 **[Live Demo](https://devquiz-app-three.vercel.app/)**

---

## ✨ Features

- 💻 **4 Language Tracks** — JavaScript, Python, HTML/CSS, and General CS
- 📊 **3 Difficulty Levels** — Beginner, Intermediate, Advanced
- ⏱️ **20-Second Timer Per Question** — auto-advances when time runs out
- 📈 **Live Progress Bar** — always know how far you are in the quiz
- ✅ **Instant Feedback** — see the correct answer immediately after each response
- 🔊 **Sound Effects** — satisfying audio cues for correct and incorrect answers
- 🏆 **Final Score Screen** — score, time taken, and a performance message
- 🔁 **Retry & Share** — retake the quiz or share your score with one click
- 📱 **Fully Responsive** — optimized for mobile, tested down to 375px width

## 🚀 Run Locally

No installation required. Just clone and open.

```bash
# Clone the repository
git clone https://github.com/RzDevCode/devquiz-app.git

# Navigate into the project
cd devquiz-app

# Open in your browser
open index.html
```

Or simply download the `index.html` file and open it in any browser.
